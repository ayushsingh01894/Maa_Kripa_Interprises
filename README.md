# 🛒 Shop Website

A responsive and user-friendly shop website built using **HTML, CSS, and JavaScript**.  
This project showcases a simple e-commerce style layout with product sections, images, and interactive elements.

🔗 **Live Demo:**  
https://ayushsingh01894.github.io/Shop-Website/

---

## 📌 Features

- Responsive design (desktop & mobile friendly)
- Clean and modern UI
- Product listing with images
- Separate pages for categories (example: cookware)
- Custom CSS styling
- JavaScript for interactivity
- Deployed using **GitHub Pages**

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout  
- **JavaScript** – Basic interactivity  
- **GitHub Pages** – Live hosting
